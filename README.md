*#Pengertian Bootstrap
Bootstrap adalah framework HTML, CSS, dan JavaScript yang digunakan untuk membuat website responsive dan modern dengan lebih cepat dan mudah.

Bootstrap menyediakan banyak komponen siap pakai seperti:
-Navbar
-Button
-Card
-Form
-Modal
-Tabel

*#Fungsi Bootstrap
-Mempercepat pembuatan website
-Membuat tampilan responsive
-Mempermudah desain UI
-Menghemat penulisan CSS
-Membantu membuat website modern

*#Kelebihan Bootstrap
✅ Mudah digunakan
✅ Responsive di semua perangkat
✅ Banyak komponen siap pakai
✅ Dokumentasi lengkap
✅ Cocok untuk pemula

*#Cara Menggunakan Bootstrap
-Tambahkan CDN Bootstrap pada HTML:
contoh:
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
Struktur Dasar Bootstrap
<div class="container">
  <h1 class="text-primary">
    Hello Bootstrap
  </h1>
</div>

*#Sistem Grid Bootstrap
-Bootstrap menggunakan sistem 12 kolom.
Contoh:
<div class="row">
  <div class="col-6">Kolom 1</div>
  <div class="col-6">Kolom 2</div>
</div>

*#Komponen Penting Bootstrap
1. Button
<button class="btn btn-primary">
  Simpan
</button>
2. Card
<div class="card">
  <div class="card-body">
    Isi Card
  </div>
</div>
3. Navbar
<nav class="navbar bg-dark navbar-dark">
4. Form
<input type="text" class="form-control">
5. Table
<table class="table">
  
*#Utility Class Bootstrap
-Margin & Padding
 1.  mt-5 → margin atas
 2.  mb-5 → margin bawah
 3.  p-5 → padding
    
-Warna Text
 1.  text-primary
 2.  text-danger
 3.  text-success
    
Responsive Bootstrap
Ukuran	  Class
Small	     -     sm
Medium     -     md
Large	     -     lg
Extra Large  - 	 xl

Contohnya:
<div class="col-md-6">
  
*#Kelemahan Bootstrap
Tampilan website kadang mirip

8#Kesimpulan
Bootstrap adalah framework yang mempermudah pembuatan website responsive dan modern dengan cepat menggunakan class HTML siap pakai. Bootstrap sangat cocok untuk pemula maupun developer web.
